## OpenAPI + Generated Client (Nx)

The backend OpenAPI spec is generated by TSOA.

### Source of truth

- `apps/backend/src/swagger/swagger.yaml`

### Single command to sync everything

From the repo root, run:

```bash
yarn openapi:sync
```

This will:

1. Generate the TSOA YAML spec
2. Copy it into `libs/api-specs/src/api-specs.openapi.yaml`
3. Regenerate `libs/app-api-client`

### CI drift check

```bash
yarn openapi:check
```

Fails if generated outputs are out of date (i.e., the repo becomes dirty after regeneration).
